SILIA is a voice- and eye-controlled Windows assistant designed for accessibility and convenience.

## Features
- Wake word: "hello windows"
- Voice commands (open apps, folders, search)
- Eye-controlled cursor movement
- Blink-to-click
- Runs automatically on Windows startup
- No manual file execution required

## Technologies Used
- Python 3.10
- MediaPipe (Face Landmarker)
- OpenCV
- SpeechRecognition
- PyAutoGUI

## Usage
1. Start Windows
2. Say "hello windows"
3. Give voice commands or switch to eye control

## Accessibility
Designed for:
- Elderly users
- Hands-free interaction
- Assistive computing scenarios
